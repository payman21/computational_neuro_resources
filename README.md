# Computational Cognitive Neuroscience Resources

Here, you'll find links to various tools, articles, datasets, and more, related to the field of Computational Cognitive Neuroscience.

## üíæ Datasets & Repositories
- [Brain Imaging Data Structure (BIDS)](https://bids.neuroimaging.io/) - A simple and intuitive way to organize and describe your neuroimaging and behavioral data.
- [BrainMap](https://www.brainmap.org/) - Database of published task and structural neuroimaging experiments with coordinate-based results.
- [Cam-CAN](https://camcan-archive.mrc-cbu.cam.ac.uk/dataaccess/) - Large-scale collaborative research project from the University of Cambridge (MRI, MEG, cognitive data).
- [Computational Psychiatry Data](https://eiko-fried.com/data/) - Data resources curated by Dr. Eiko Fried.
- [CRCNS](https://crcns.org/) - Collaborative Research in Computational Neuroscience - Data sharing.
- [DANDI Archive](https://dandiarchive.org/) - Archive for publishing and sharing neurophysiology data (electrophysiology, optophysiology, etc.).
- [DRYAD](https://datadryad.org/stash) - Open data publishing platform.
- [EBRAINS](https://search.kg.ebrains.eu/?category=Dataset) - Platform to share models, tools, and data.
- [Google Dataset Search](https://datasetsearch.research.google.com/)
- [Healthy Brain Network](https://brain.labsolver.org/hbn.html) - Large-scale MRI dataset.
- [Human Brain Project](https://www.humanbrainproject.eu/en/)
- [International Brain Laboratory (IBL)](https://www.internationalbrainlab.com/) - Releases all data sets within 12 months of collection.
- [Neuro Data Without Borders (NWB)](https://www.nwb.org/) - Data standard for neurophysiology.
- [Neurosynth](https://neurosynth.org/) - Platform for large-scale, automated synthesis of fMRI data.
- [NeuroVault](https://neurovault.org/) - A public repository of unthresholded statistical maps, parcellations, and atlases.
- [OpenNEURO](https://openneuro.org/) - A free platform for sharing BIDS-compliant MRI, PET, MEG, EEG, and iEEG data.
- [Poldracklab Software and Data](https://poldracklab.org/software.html)
- [The fMRI Data Center](https://www.nitrc.org/include/about_us.php) - A free database of fMRI data used in peer-reviewed studies.
- [THINGS-data](https://plus.figshare.com/collections/THINGS-data_A_multimodal_collection_of_large-scale_datasets_for_investigating_object_representations_in_brain_and_behavior/6161151) - Multimodal collection of large-scale datasets for investigating object representations.
- [Zenodo](https://zenodo.org/) - Open Science platform for data sharing.
- [3D Brain Atlas](https://neurotorium.org/tool/brain-atlas/) - Interactive brain atlas.

## üõ†Ô∏è Software, Code & Tools

### Analysis & Processing
- [Brain Connectivity Toolbox (MATLAB)](https://sites.google.com/site/bctnet/) - MATLAB toolbox for complex brain-network analysis.
- [Brain Connectivity Toolbox (Python)](https://pypi.org/project/bctpy/) - Python port of the BCT.
- [CorticalParcellation](https://freesurfer.net/fswiki/CorticalParcellation_DU15NET) - A novel 15-network parcellation of the cerebral cortex.
- [DeepPrep](https://github.com/indilab/deepprep) - Accelerated, scalable, and robust pipeline for neuroimaging preprocessing.
- [ENIGMA TOOLBOX](https://github.com/MICA-MNI/ENIGMA) - For accessing ENIGMA-derived statistical maps and visualizing cortical/subcortical data.
- [fMRIPrep](https://fmriprep.org/en/stable/) - A Robust Preprocessing Pipeline for fMRI Data.
- [FreeSurfer](https://surfer.nmr.mgh.harvard.edu/) - Toolkit for processing, analyzing, and visualizing human brain MR images.
- [Functio-Structural Current (FSC)](https://github.com/vilsaira/FSC) - Python code to combine functional and structural brain connectivity.
- [JuSpace](https://github.com/juryxy/JuSpace) - Integration of imaging modalities with PET-derived neurophysiological measures.
- [nemos](https://github.com/flatironinstitute/nemos) - Statistical modeling framework for systems neuroscience (GLM for spikes).
- [netneurotools](https://github.com/justinehansen/netneurotools) - Functions for network neuroscience analysis.
- [NetworkX](https://networkx.org/documentation/stable/index.html) - Python package for complex networks.
- [neuromaps](https://github.com/netneurolab/neuromaps) - Toolbox for making statistically-rigorous comparisons between brain maps.
- [NLTools](https://nltools.org/) - Python package for analyzing neuroimaging data.
- [pynapple](https://pynapple-org.github.io/pynapple/) - Lightweight Python library for neurophysiological data analysis.
- [torch_brain](https://github.com/neuro-galaxy/torch_brain) - Python library for various deep learning models designed for neuroscience.

### Modeling & Simulation
- [Brain Dynamics Toolbox](https://bdtoolbox.org/) - MATLAB software for simulating bespoke dynamical systems.
- [BrainPy](https://github.com/brainpy/BrainPy) - Flexible framework for computational neuroscience and brain-inspired computation.
- [Human Neocortical Neurosolver (HNN)](https://hnn.brown.edu/) - Test hypotheses on circuit mechanisms underlying EEG/MEG data.
- [NEST](https://www.nest-simulator.org/) - Simulator for spiking neural network models.
- [NeuroGym](https://github.com/neurogym/neurogym) - Curated collection of neuroscience tasks for training neural networks.
- [The Virtual Brain](https://www.thevirtualbrain.org/tvb/zwei) - Brain simulation platform.
- [WhoBPyT](https://github.com/GriffithsLab/whobpyt) - Whole-Brain Modelling in PyTorch.
- [fast-slow brain network model](https://github.com/BrainModes/fast-slow/) - Code from *Learning how network structure shapes decision-making*.
- [Heterogenous Neural Model (HNM)](https://github.com/KevinAquino/HNM) - For large scale modelling of brain activity.

### Visualization & Platforms
- [Connectome Workbench](https://www.humanconnectome.org/software/connectome-workbench) - Visualization and discovery tool for neuroimaging data.
- [fastplotlib](https://github.com/fastplotlib/fastplotlib) - High-performance GPU plotting library for scientific visualization.
- [MRIcroGL](https://www.nitrc.org/projects/mricrogl) - View 2D slices and renderings of brain imaging data.
- [Neurodesk](https://www.neurodesk.org/) - A flexible and scalable data analysis environment (containerized).
- [Subcortex visualization software](https://anniegbryant.github.io/subcortex_visualization/) - Generate two-dimensional subcortex images.
- [brainlife](https://brainlife.io/about/) - Open-source, secure reproducible neuroscience analysis platform.
